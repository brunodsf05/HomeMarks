# HomeMarks

Do you want your **homepage** to show your **bookmarks**? This is what you’re
looking for!

Enhance the bookmark experience with an attractive, fast, and functional
interface, designed to access everything instantly.

![HomeMarks extension in action](/docs/assets/hero.png)

## Features

- No configuration needed... HomeMarks will always load your browser bookmarks!
- Clean and minimalist interface.
- Fast navigation using either keyboard or mouse.
- Search bar to find bookmarks or perform web searches.
- Compatible with multiple browsers.

> [!WARNING]
> This project is still under development. Firefox is not supported yet. At the
> moment, HomeMarks shows your bookmarks, allows filtering, and enables web
> searches.

## Development

### Installation

1. Clone this repository and enter the project.
2. Install dependencies with `pnpm install`.

### Test the interface

1. Run `pnpm run dev`. This will start a development server; the displayed
   bookmarks are **MockData**.

### Test the extension

#### Requirements

A **Chromium-based** browser. (**Firefox** compatibility is under development.)

1. Run `pnpm run build`.
2. Open the **extensions management** page.
3. Enable developer mode.
4. Select **load unpacked**.
5. Choose the **dist** folder (generated by `pnpm run build`).
6. Open a new tab.
